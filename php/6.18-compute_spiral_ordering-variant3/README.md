Write a program to enumerate the first n pairs of integers (a, b) in spiral order, starting from (0, 0) followed by
(1, 0).  For example, if n = 10, your output should be (0, 0), (1, 0), (1, -1), (0, -1), (-1, -1), (-1, 0), (-1, 1),
